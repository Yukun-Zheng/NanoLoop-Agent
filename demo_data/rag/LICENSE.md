# 知识资产许可与署名说明

`demo_data/rag/sources/` 中的正文由 NanoLoop Agent 项目为竞赛演示重新撰写，按本仓库
`pyproject.toml` 所述的竞赛开发条款使用。知识卡没有捆绑或复制论文全文。

每张知识卡末尾列出的外部论文保留其原始作者、题名、期刊、DOI 和开放许可：

- Nature Communications 来源：CC BY 4.0；
- Nanoscale Advances 来源：CC BY 3.0。

知识卡是对开放来源的概括与项目使用边界说明，不改变原始作品许可。使用或扩展知识卡时：

1. 保留对应参考文献和 DOI；
2. 不把项目样品标签自动改写为未经确认的化学式；
3. 不把文献中的一般规律表述为当前样品已经证实的因果结论；
4. 新增外部全文前，必须另行核对许可、文件 SHA-256 和演示授权。
